  UNDEFINED INDEX ERROR - FIXED
================================================================================

ERROR: Notice: Undefined index: member_id in loans.php on line 15
STATUS: ✅ FIXED

================================================================================
WHAT WAS WRONG
================================================================================

The code was trying to access $_SESSION['member_id'] without checking if it exists.

This error occurred because:
- Admin users don't have a member_id in their session
- The code assumed all logged-in users would have member_id
- When an admin tried to access a member page, the error appeared

================================================================================
HOW IT WAS FIXED
================================================================================

Added proper checks before accessing $_SESSION['member_id']:

// Check if user is admin (redirect admins away)
if (isset($_SESSION['user_role']) && $_SESSION['user_role'] === 'admin') {
    redirect('admin.php');
}

// Check if member_id exists (redirect if missing)
if (!isset($_SESSION['member_id']) || empty($_SESSION['member_id'])) {
    redirect('login.php');
}

// Now safe to use member_id
$member_id = $_SESSION['member_id'];
