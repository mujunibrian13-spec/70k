================================================================================
                        HOW INTEREST WORKS
================================================================================

The System Logic:

When a loan is taken:
  1. Member borrows money from group fund
  2. Interest is calculated (2% monthly)
  3. When payment is made, interest is added to pool

Interest Distribution:
  1. At end of month, all interest is distributed
  2. Distribution happens to ALL active members
  3. Share is based on MEMBERSHIP, not just savings
  4. Even members with 0 savings earn interest

Example Scenario:

  Group has 20 members
  Total loans outstanding: 50,000,000 UGX
  Monthly interest earned: 1,000,000 UGX (2%)
  
  Distribution to members (if equally):
  1,000,000 ÷ 20 = 50,000 UGX per member
  
  Even if member has no savings, they get 50,000 UGX interest!

================================================================================
                    TECHNICAL IMPLEMENTATION
================================================================================

Member Card (Card 1):
  Query: Searches members table for names containing "alison" or "alson"
  Retrieves:
    - Member ID, name, email, savings_amount
    - Total interest from member_interest table
    - Total loans from loans table
  Updates: Real-time from database

Interest Distribution Card (Card 2):
  Query: Sums all interest_amount from member_interest table
  Shows: Total interest ever distributed to all members
  Updates: Automatically when new interest is distributed

Interest Sharing Card (Card 3):
  Query: Counts distinct members in member_interest table with interest > 0
  Shows: Number of members who have earned interest
  Purpose: Dashboard awareness of member participation

================================================================================
                      DATABASE QUERIES USED
================================================================================

Query 1 - Find Member:
  SELECT id, full_name, email, savings_amount FROM members 
  WHERE full_name LIKE '%alison%' OR full_name LIKE '%alson%'

Query 2 - Member Interest:
  SELECT COALESCE(SUM(interest_amount), 0) as total 
  FROM member_interest WHERE member_id = ?

Query 3 - Member Loans:
  SELECT COALESCE(SUM(loan_amount), 0) as total 
  FROM loans WHERE member_id = ? 
  AND status IN ('active', 'completed', 'approved')

Query 4 - Total Interest Distributed:
  SELECT COALESCE(SUM(interest_amount), 0) as total 
  FROM member_interest

Query 5 - Members with Interest:
  SELECT COUNT(DISTINCT member_id) as count 
  FROM member_interest WHERE interest_amount > 0
