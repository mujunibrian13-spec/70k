HOW IT WORKS
================================================================================

MEMBER SELF-SERVICE PASSWORD CHANGE:

Step 1: Access
  - Member logs in
  - Clicks name dropdown in top right
  - Selects "Change Password"
  - Goes to change_password.php

Step 2: Form
  - Displays form with 3 fields:
    * Current password (verification)
    * New password (6+ characters)
    * Confirm new password
  - Shows password requirements
  - Shows security tips

Step 3: Submit
  - Member enters information
  - Clicks "Change Password"
  - Form submits via POST

Step 4: Validation
  - Current password verified
  - New password length checked (6+)
  - Confirmation matches new password
  - New password different from current
  - All fields required

Step 5: Update
  - Password hashed using SHA-256
  - Stored in users table
  - Updated_at timestamp set
  - Transaction logged

Step 6: Result
  - Success message displays
  - Member logs out
  - Logs back in with new password

---

ADMIN PASSWORD RESET:

Step 1: Access
  - Admin logs in
  - Clicks "Reset Password" in navbar
  - Goes to admin_reset_member_password.php

Step 2: Select Member
  - Dropdown list of all members
  - Admin selects member
  - Member details display:
    * Username
    * Email
    * Phone
    * Status

Step 3: Enter Password
  - Admin enters new password (6+ chars)
  - Admin confirms password
  - Passwords must match

Step 4: Confirm
  - Admin clicks "Reset Password"
  - Confirmation dialog appears
  - Admin clicks OK to confirm
  - (Prevents accidental resets)

Step 5: Update
  - Password hashed
  - Stored in users table
  - Updated_at timestamp set
  - Transaction logged

Step 6: Communicate
  - Success message shows
  - Admin communicates password to member securely
  - Member logs in with new password
  - Member changes password if desired
