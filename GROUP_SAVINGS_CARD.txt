================================================================================
                         WHAT IT SHOWS
================================================================================

The card displays four key pieces of information:

1. TOTAL GROUP SAVINGS
   Label: "Total Group Savings"
   Value: Sum of all members' savings (in UGX)
   Color: Green
   
   Example: 10,500,000 UGX

2. ALIGANYIRA ALSON'S SAVINGS
   Label: "Aliganyira Alson's Savings"
   Value: Alison's individual savings (in UGX)
   Color: Blue/Purple
   
   Example: 500,000 UGX

3. OTHER MEMBERS COMBINED (CALCULATED)
   Label: "Other Members Combined"
   Value: Total Savings - Alison's Savings
   Color: Teal (highlighted)
   
   Example: 10,500,000 - 500,000 = 10,000,000 UGX

4. ALISON'S PERCENTAGE
   Label: "Info line"
   Value: Alison's contribution as % of total
   
   Example: "Alison: 4.8% of total"

================================================================================
                          HOW IT CALCULATES
================================================================================

Formula:
  Other Members Savings = Total Group Savings - Alison's Savings

Example Scenario:
  Total Group Savings:      10,500,000 UGX
  Alison's Savings:             500,000 UGX
  ────────────────────────────────────
  Other Members Combined:   10,000,000 UGX
  
  Alison's Percentage:
    (500,000 / 10,500,000) × 100 = 4.8% of total

================================================================================
                       TECHNICAL DETAILS
================================================================================

Data Source:
  - Total Savings: From $total_savings (calculated earlier in admin.php)
  - Alison Savings: From $alison_savings (fetched from members table)

Calculation:
  $other_members_savings = $total_savings - $alison_savings

Percentage Calculation:
  $alison_percentage = ($alison_savings / $total_savings) × 100
  Formatted to 1 decimal place using number_format()

Display Format:
  All amounts formatted using formatCurrency() function
  Percentage shown with 1 decimal place

Error Handling:
  If total_savings = 0: Shows "No savings recorded"
  If no member found: Uses $alison_savings = 0
  Graceful calculation without division errors
