HOW IT WORKS
================================================================================

MEMBER WORKFLOW:
1. Go to Pay Loan page
2. Submit payment (amount, method, receipt, notes)
3. See message: "Payment submitted! Pending admin approval"
4. See payment in "Pending Approvals" section
5. Wait for admin approval
6. Once approved, see updated loan balance

ADMIN WORKFLOW:
1. Login to Admin Dashboard
2. See "Payment Approvals" stat card with count
3. Click card or "Approve Payments" in navigation
4. View list of pending payments
5. Click "Approve" or "Reject" for each payment
6. Confirmation dialog appears
7. Payment processed:
   - If Approved: Loan updated, interest distributed
   - If Rejected: No changes, member can resubmit

AUTOMATIC PROCESSING:
When admin approves payment:
✓ Loan balance updated
✓ Amount paid updated
✓ Loan status set to 'active' or 'cleared' (if fully paid)
✓ Interest calculated (payment amount × 2%)
✓ Interest distributed to all members by savings ratio
✓ Transactions logged for audit trail
